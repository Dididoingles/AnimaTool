Sobre o Projeto
Animation Studio PRO e uma ferramenta completa de animacao que roda 100% no navegador, sem necessidade de instalacao. Com ela voce pode criar animacoes complexas com multiplas camadas, efeitos 3D, keyframes e curvas de easing, tudo em uma interface profissional e intuitiva.

Para que serve?
Motion Graphics - Animacoes para videos institucionais

Conteudo para Redes Sociais - Stories, reels e posts animados

Apresentacoes - Elementos animados para slides

Thumbnails - Imagens com movimento para YouTube

Prototipagem - Animacoes de UI/UX

Videos Educacionais - Conteudo didatico animado

Como Usar
Instalacao
Baixe o arquivo AnimationStudio_HD.html

Salve em qualquer pasta do seu computador

Abra com qualquer navegador moderno (Chrome, Edge, Firefox, Opera)

Pronto! Nao precisa de instalacao, servidor ou internet (exceto para Google Fonts)

Primeiros Passos
1. Criando sua primeira animacao
text
1. Clique em "Nova Camada" -> "Adicionar Texto"
2. De duplo clique no texto no canvas para editar
3. Arraste o texto para a posicao inicial
4. Mova o playhead (linha amarela) para 2 segundos
5. Arraste o texto para outra posicao
6. Clique em "Adicionar Keyframe"
7. Clique em "Play" para ver a animacao!
2. Adicionando uma imagem
text
1. Clique em "Nova Camada" -> "Importar Midia"
2. Selecione uma imagem PNG, JPG ou GIF
3. A imagem aparecera no canvas
4. Repita o processo de keyframes
3. Aplicando efeito 3D
text
1. Selecione uma camada
2. Na aba "Perspectiva 3D" (sidebar direita)
3. Ajuste os sliders:
   - Rotate X: inclina para cima/baixo
   - Rotate Y: inclina para esquerda/direita
   - Profundidade Z: aproxima/afasta
4. Adicione keyframes para animar a perspectiva
4. Gravando o resultado
text
1. Escolha a qualidade (Alta/Media/Baixa)
2. Clique em "Gravar Tela"
3. Na janela que abrir, selecione a ABA ATUAL
4. Clique em "Compartilhar"
5. A animacao comecara a tocar e gravar
6. Clique em "Parar Gravacao" quando terminar
7. O video sera baixado automaticamente

Tutoriais
Tutorial 1: Texto que entra voando
text
1. Adicione uma camada de texto
2. Posicione o texto fora da tela (X = -300)
3. Adicione keyframe (tempo 0s)
4. Mova playhead para 2s
5. Posicione texto no centro (X = 0)
6. Adicione keyframe
7. Selecione easing "Bounce" na propriedade Posicao
8. Play!
Tutorial 2: Efeito de zoom 3D
text
1. Adicione uma imagem
2. Vá para Perspectiva 3D
3. Tempo 0s: Profundidade Z = -200
4. Adicione keyframe
5. Tempo 2s: Profundidade Z = 200
6. Adicione keyframe
7. Selecione easing "Elastic"
8. Play!
Tutorial 3: Rotacao 360 graus
text
1. Selecione uma camada
2. Vá para Rotacao Z
3. Tempo 0s: 0 graus
4. Adicione keyframe
5. Tempo 2s: 360 graus
6. Adicione keyframe
7. Selecione easing "Linear"
8. A imagem vai girar completamente
Tutorial 4: Animacao com video
text
1. Importe um video MP4
2. Ajuste duracao para 5s
3. Tempo 0s: Escala = 0.5
4. Adicione keyframe
5. Tempo 3s: Escala = 1.5
6. Adicione keyframe
7. Selecione easing "Bounce"
8. O video vai "saltar" enquanto toca

Dicas Profissionais
Use easing! Bounce da efeito elastico, Elastic da overshoot

Combine propriedades - Posicao + Rotacao + Escala juntos criam movimentos complexos

Keyframes proximos = movimento rapido; distantes = movimento lento

Sempre adicione keyframe no inicio (0s) para ter controle total

Use o botao "Atualizar Keyframe" se quiser ajustar um movimento existente

Para animacoes suaves, use curvas Ease ou Smooth

Grave em Alta qualidade apenas se precisar de 1080p (arquivos maiores)

Videos importados ficam mudos para permitir autoplay

Use o botao de zoom na timeline para keyframes muito proximos

Renomeie suas camadas para organizar projetos complexos

Solucao de Problemas
O video nao aparece ao importar
Aguarde o carregamento (imagens grandes podem demorar)

Verifique se o formato e suportado

Tente converter o video para MP4 (H.264)

O keyframe nao fixa
Sempre clique em "Adicionar Keyframe" apos mover o elemento

Use o botao "Atualizar Keyframe" para modificar um existente

Verifique se o playhead esta na posicao desejada

A gravacao nao comeca
Clique em "Gravar Tela" e autorize a permissao

Selecione a ABA ATUAL (nao a janela inteira)

Clique em "Compartilhar" no popup do navegador

O video gravado esta com baixa qualidade
Acesse as configuracoes e escolha "Alta" na qualidade

Verifique se o navegador suporta VP9

O texto nao aparece
De duplo clique no texto no canvas

Digite o novo texto e confirme

Certifique-se que a camada esta visivel (icone de olho)

A animacao nao toca
Clique em "Play" ou arraste o playhead manualmente

Verifique se ha keyframes adicionados

A duracao da animacao pode ser muito curta
